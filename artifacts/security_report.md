# Отчёт по безопасности

## Область проверки

Проверка относится к полностью синтетическому локальному MVP. Реальные банковские
данные, OAuth, корпоративные ACL и внешние записи не использовались.

## Подтверждённые точечные проверки

На фокусном тестовом прогоне прошли проверки:

- выявление prompt injection в исходном документе;
- выявление secret-like поля и значения;
- редактирование email и телефона;
- блокирование ресурса вне разрешённой области;
- блокирование кредитного решения даже при A4/approval;
- отказ approved-выполнения без receipt;
- отсутствие внешних записей в dossier executor;
- блокирование неизвестной стоимости и превышения бюджетных лимитов.

Сохранённый E2E baseline сообщает:

- 1 запрещённое событие заблокировано — расчёт на синтетике;
- `pii_leakage_rate = 0.0` — результат валидатора синтетического trace, а не замер
  production DLP;
- `external_writes = []` — mock-выполнение;
- demo-runtime cost = 0 USD — только локальный детерминированный ledger.

## Неподтверждённые контуры

- полный secret/PII scan репозитория, отчётов, PDF metadata, скриншотов и видео;
- supply-chain и licence scan;
- контейнерная или процессная изоляция sandbox;
- production RBAC/DLP и отзыв OAuth;
- production identity and clock source for approval receipts;
- реальные MCP-коннекторы.

Focused suite 42/42 зелёная. Итоговый security sign-off остаётся **pending** до
выполнения whole-pack, supply-chain, PDF и video проверок.
