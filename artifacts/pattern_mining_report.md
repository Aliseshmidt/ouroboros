## Метод

Алгоритм группирует события по `correlation_id`, исключает помеченный шум,
заблокированные события и дубли retry, сравнивает последовательности детерминированной
метрикой порядка и пересечения, затем формирует кластеры. Ground truth используется
после поиска только для оценки.
