# Отчёт по поиску паттернов

## Данные

Источник — полностью синтетический trace за 10 рабочих дней. Свежий baseline содержит
179 событий; три чистых прогона дали одинаковый report hash.

## Алгоритм

События группируются по `correlation_id`. Помеченный шум, blocked/cancelled и retry-
дубли не участвуют в sequence. Последовательности сравниваются детерминированной
комбинацией ordered similarity и пересечения действий, после чего кластеризуются.
Stable и variable actions рассчитываются из поддержки шагов внутри кластера.

Ground truth не передаётся miner: она используется только после расчёта для метрик.

## Вариативность

Для flagship `mail:search_correspondence` определён как variable action; остальные
десять действий стабильны. Noise correlations исключены. Точечные тесты вариантов,
шума, похожих последовательностей и недостаточной частоты прошли.

## Ограничения

- synthetic ground truth проще production trace;
- confidence — heuristic score, не калиброванная вероятность;
- названия трёх известных семейств задаются детерминированным rule mapping после
  кластеризации;
- focused suite 42/42 и clean-run determinism test зелёные;
- production validation отсутствует; browser/video evidence подтверждает только работу MVP.
