# Personal Evolution Engine

Универсальный агент обнаруживает повторяющиеся персональные процессы,
оценивает эффект, собирает микронавык, проверяет его на обезличенной истории и
оставляет любые внешние изменения за явным подтверждением сотрудника.

## Запуск за четыре шага

1. Запустите Ouroboros: `ouroboros server`.
2. В **Skills** выполните Review для `personal_evolution_engine`, затем включите навык.
3. В чате попросите: «Создай demo trace, проанализируй паттерны и предложи микронавык для лучшего паттерна».
4. После проверки 10+ примеров подтвердите точной фразой `APPROVE <proposal_id>` и попросите выполнить approved draft.

Для автономной проверки без UI:

```bash
.venv/bin/python skills/personal_evolution_engine/scripts/demo.py
```

Успешный результат содержит `patterns: 3`, `verified_cases: 12`,
`verification_status: passed`, `approval_status: approved` и
`external_writes: []`.

## Что демонстрировать

- 36 обезличенных процессов и 144 события;
- три найденных паттерна для разных типов офисной работы;
- прозрачный расчёт времени AS IS - TO BE;
- автоматически собранный микронавык и его autonomy policy;
- 12/12 пройденных исторических проверок;
- approval gate, audit и feedback-driven evolution;
- виджет **Personal Evolution** с портфелем метрик.

Прототип не хранит исходный цифровой след и не вызывает внешние системы.
Production-запись разрешается только через прошедший ревью MCP-коннектор,
после показа diff и нового подтверждения пользователя.
